# Singly Linked List in C++

This project implements a singly linked list in C++ using a head pointer without a dummy header node. It demonstrates fundamental linked list operations and manual memory management.

## Features

- Node insertion and removal
- Node retrieval and search
- List traversal
- Size tracking
- Empty check
- Clear function
- Copy constructor and assignment operator
- Proper destructor for memory cleanup

## Design

This implementation uses a minimal structure (no dummy header node), requiring explicit handling of edge cases such as insertion and deletion at the head of the list.

## Testing

A `driver.cpp` file is included to test correctness of all list operations. A `compare_driver.cpp` is also included to compare performance against a circular doubly linked list implementation.

## Purpose

This project was developed to reinforce understanding of pointer-based data structures, memory management, and linked list fundamentals in C++.
